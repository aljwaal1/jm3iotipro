# جمعيتي Pro v2

تطبيق إدارة الجمعيات الشهرية بالعربية مع كشوفات PDF وصور PNG.

## المميزات
- إدارة جمعيات متعددة
- تتبع الدفعات لكل شهر
- **كشف حساب PDF** قابل للمشاركة
- **تصدير صورة PNG** للكشف
- تذكير عبر واتساب
- نسخ احتياطي JSON
- حماية بكلمة مرور
- يعمل بدون إنترنت

## التثبيت
حمّل APK من مجلد `apk/`
